# Decisions: Delano Viewer Redesign

## 2026-05-08: Sidebar Is Workspace-Level Navigation

Decision: the left sidebar should not primarily mirror the selected project's dashboard sections. It should expose workspace-level views for open work, progress, validation, warnings, and blockers. The selected project remains visible through the project selector and main-pane project overview.

Rationale: duplicating the same Current Work, Blockers, Progress, Validation, and Warnings structure in both the sidebar and the selected project's Overview creates unclear scope. Workspace navigation makes the sidebar useful across projects while the main pane remains project-specific.

Status: accepted for the current redesign iteration.
