# Delano Skills

Handbook-aligned skill contracts.

Core workflow skills:

- `discovery-skill`
- `planning-skill`
- `breakdown-skill`
- `sync-skill`
- `execution-skill`
- `quality-skill`
- `closeout-skill`
- `learning-skill`

Each skill defines:
- intent and trigger context
- required inputs
- output schema
- quality checks
- failure behavior
- allowed side effects
- script hooks
